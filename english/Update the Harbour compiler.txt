1. Download Harbour's Nightly version from this link:
   https://sourceforge.net/projects/harbour-project/files/binaries-windows/nightly/harbour-nightly-win.exe/download
2. Install Harbour to C:\OOHG\HB32
3. At folder C:\OOHG\SOURCE create a text file named CLEAN.BAT with the following content:
      SET PATH=C:\OOHG
      SET HG_HRB=
      SET HG_MINGW=
      SET HG_ROOT=
      SET LIB_GUI=
      SET LIB_HRB=
      CALL MAKELIB.BAT HB32
4. Open a Command Prompt at folder C:\OOHG\SOURCE and execute CLEAN.BAT
